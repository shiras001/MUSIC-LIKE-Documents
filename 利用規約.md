# 利用規約

最終更新日：2026年2月16日

## 第1条（目的）
本規約は、LIFE LIKE（以下「当社」といいます。）が提供するAndroid向け音楽プレーヤーアプリ（以下「本アプリ」といいます。）の利用条件を定めるものです。本アプリは、利用者がローカルに保存された音声ファイルや歌詞ファイルを読み込み、再生・表示する機能、ローカルの画像を壁紙として表示する機能、読み込んだ音声ファイルのメタデータや歌詞ファイルの編集機能、通知・ロック画面でのプレーヤー表示機能を提供します。

## 第2条（定義）
本規約において使用する用語の定義は、以下の通りとします。
1.  「利用者」とは、本規約に同意の上、本アプリを利用する個人または法人をいいます。
2.  「コンテンツ」とは、利用者が本アプリに読み込む音声ファイル、歌詞ファイル、およびローカル画像ファイルをいいます。
3.  「有料プラン」とは、当社が提供する月額課金制のサービスプランをいいます。
4.  「無料プラン」とは、広告表示を伴う無償のサービスプランをいいます。
5.  「広告」とは、無料プランにおいて表示されるネイティブ広告、インタースティシャル広告、リワード広告をいいます。

## 第3条（本アプリの提供）
1.  当社は、利用者に本アプリを提供します。
2.  本アプリは、ローカルに保存された音声ファイル、歌詞ファイル、およびローカル画像を読み込み、再生・表示する機能を提供します。
3.  本アプリは、読み込んだ音声ファイルのメタデータおよび歌詞ファイルの編集機能を提供します。
4.  本アプリは、通知領域およびロック画面にプレーヤーを表示する機能を提供します。
5.  当社は、利用者の利用状況に関する情報を収集します。収集する情報の詳細については、別途定めるプライバシーポリシーに従うものとします。

## 第4条（利用プラン）
1.  本アプリには、月額有料プランと無料プランがあります。
2.  無料プランでは、ネイティブ広告、インタースティシャル広告、リワード広告が表示されます。
3.  リワード広告を視聴することにより、特定の機能の利用が可能となる場合があります。その詳細については、アプリ内での案内によるものとします。
4.  有料プランの料金、提供される機能、およびその他の詳細については、別途当社が定める料金表および説明によるものとします。
